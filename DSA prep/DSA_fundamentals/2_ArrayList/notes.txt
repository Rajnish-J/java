ArrayList:
It is a dynamic array of objects.

Difference between Arrae and ArrayList:

1. Arrays are natural data structures whereas Arraylist is an added class
2. Arrays are of fixed size whereas Arraylist can grow and shrink
3. Arrays can hold primitive data types whereas Arraylist can only hold object

Creating an Arraylist

The object type must be specified using <> when creating an Arraylist
Ex: Arraylist<String> list 1 = new ArrayList<>()
    ArrayList<lnteger> list2 = new ArrayList<>()

Note: Notice the Inte wrapper class used in place of primitive data type
